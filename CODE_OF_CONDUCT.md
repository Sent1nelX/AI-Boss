# Code of Conduct

AI-Boss Super Brain придерживается простого правила: обсуждаем идеи и код жёстко, людей — бережно.

## Ожидаемое поведение

- Уважительное общение.
- Конкретная критика с примерами.
- Готовность объяснять решения новичкам.
- Внимание к безопасности пользователей и их данных.

## Недопустимое поведение

- Оскорбления, harassment, дискриминация.
- Публикация секретов, токенов, приватных ключей или персональных данных.
- Подталкивание к опасному поведению: auto-push/deploy без согласия, обход git-safety, удаление пользовательских данных.

Maintainers могут закрывать issues/PRs и ограничивать участие при нарушении этих правил.
