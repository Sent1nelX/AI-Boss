# Security Policy

AI-Boss Super Brain управляет локальными AI CLI-инструментами и может запускать Codex для изменения git-проектов. Поэтому любые уязвимости в маршрутизации, web UI, git-safety проверках и работе с путями считаются важными.

## Поддерживаемая версия

Проект находится в активной разработке. Security fixes принимаются в ветку `main`.

## Как сообщить о проблеме

Если вы нашли проблему безопасности:

1. Не публикуйте exploit публично.
2. Создайте private security advisory в GitHub, если доступно.
3. Если advisory недоступен, откройте issue без exploit details и укажите, что готовы передать детали приватно.

## Важные границы безопасности

- Web UI рассчитан только на `127.0.0.1`.
- Не публикуйте `ai-boss web` через reverse proxy, tunnel или внешний bind без отдельной защиты.
- AI-Boss не должен хранить API keys, токены и приватные ключи.
- AI-Boss не должен делать auto-push, auto-deploy или auto-merge.
